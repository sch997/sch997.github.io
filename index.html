Publications / Manuscripts

LTFE: Learning Layered Transparent Frame Extrapolation for Real-Time Rendering
Wei Liao, Changhao Song, Youcheng Cai, Ligang Liu
Submitted to IEEE/CVF Conference on Computer Vision and Pattern Recognition (CVPR)
We propose the first deep learning frame extrapolation framework for transparent objects using an onion-like layered architecture inspired by OIT. I contributed by designing a Mask-guided Refinement module to optimize details in transparent regions and implementing backward warping with atomic minimum operations to resolve pixel collisions. I also led the dataset construction (UE5) and quantitative evaluation (PSNR/SSIM).
